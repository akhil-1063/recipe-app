# recipe-app

A simple recipe browsing app built with React, TypeScript, and Vite.

## Features

- Browse recipes fetched from [TheMealDB](https://www.themealdb.com/api.php)
- Search recipes by name
- Filter by category
- Click any recipe card to view full details including ingredients, instructions, and a YouTube link

## Tech Stack

- React + TypeScript
- Vite
- Axios

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
